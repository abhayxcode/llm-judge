# @llm-judge/web

Next.js 15 app. Will host traces dashboard, datasets, runs, annotation UI, metric pages with Bias Reports.

## M1 status

Skeleton stub page with links to local services. Real UI in M3+.

## Run locally

```bash
pnpm install
pnpm --filter @llm-judge/web dev
```

Open http://localhost:3000.

## Build

```bash
pnpm --filter @llm-judge/web build
```
